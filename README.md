## Лабораторна робота №3 — Класи в Java
### Тема
Робота з класами у мові Java. Створення власних класів, реалізація інтерфейсу Comparable, сортування та пошук об’єктів.
Виконала: Дубенок Діана  
Група: ІМ - 34    
Варіант №4: клас Одяг
### Завдання
Створити клас Clothing, що містить не менше 5 полів.
У виконавчому методі створити масив об’єктів цього класу.
Відсортувати масив:
за одним полем — за зростанням;
за іншим — за спаданням, використовуючи стандартні засоби сортування (Arrays.sort()).
Знайти в масиві об’єкт, ідентичний заданому (введеному користувачем).
Детально задокументувати код.
